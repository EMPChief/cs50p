Goal:
    Convert text emoticons into emojis.

Input:
    - Text string
      Examples: "Hello :)", "I am sad :(", "Hi :) :("

Output:
    - Modified text with emojis
      ":)" → "🙂"
      ":(" → "🙁"

Process / Steps:
    1. Receive text input from user
    2. Replace emoticons:
        - Replace ":)" with "🙂"
        - Replace ":(" with "🙁"
    3. Display updated text

Example:
    Hello :)        → Hello 🙂
    I am sad :(     → I am sad 🙁
    Hi :) :(        → Hi 🙂 🙁
