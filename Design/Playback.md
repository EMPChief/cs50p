Goal:
    Replace all spaces in a text with ellipses ("...").

Input:
    - Text string
      Examples: "Hello world", "Python is fun", "I am here"

Output:
    - Modified text with spaces replaced by "..."

Process / Steps:
    1. Receive text input from user
    2. Replace all spaces with "..."
    3. Display updated text

Example:
    Hello world       → Hello...world
    Python is fun     → Python...is...fun
    I am here         → I...am...here
