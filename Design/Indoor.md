Goal:
    Convert text to lowercase (indoor voice).

Input:
    - Text string
      Examples: "HELLO", "Hi There", "WHAT'S UP?"

Output:
    - Lowercase text

Process / Steps:
    1. Receive text input from user
    2. Convert text to lowercase
    3. Display result

Example:
    HELLO        → hello
    Hi There     → hi there
    WHAT'S UP?   → what's up?
